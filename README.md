# Harper for BBEdit

A color scheme inspired by the art of [Charley Harper](http://en.wikipedia.org/wiki/Charley_Harper) (1922-2007)

![Screenshot](https://raw.github.com/idleberg/Harper-BBEdit/master/images/screenshot.png)

## Installation

1. Download the color scheme to `~/Application Support/BBEdit/Color Schemes`
2. Choose the scheme from the BBEdit preferences menu
